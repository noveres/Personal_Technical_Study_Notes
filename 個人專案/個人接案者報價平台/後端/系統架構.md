| 層級 / 模組               | 說明                               | 主要內容                                                                                |
| --------------------- | -------------------------------- | ----------------------------------------------------------------------------------- |
| domain                | 領域層，負責核心業務規則與模型                  | Entity、Aggregate Root、Value Object、Domain Service、Repository Interface、Domain Event |
| domain_shared         | 領域共用層，提供跨模組共用定義                  | Enum、常數、錯誤碼、權限名稱、共用型別                                                               |
| application           | 應用服務層，負責協調用例與流程                  | Application Service、Command / Query、交易流程、DTO 轉換                                     |
| application_contracts | 應用合約層，定義對外介面與資料結構                | Request DTO、Response DTO、Service Interface、查詢條件、分頁模型                                |
| infrastructure        | 基礎設施層，負責技術實作                     | Repository 實作、ORM、外部服務介接、檔案儲存、快取、訊息佇列                                               |
| web                   | Web 表示層，負責 HTTP/API 對外入口，不包含業務邏輯 | Controller、統一回應格式、全域例外處理、驗證錯誤處理、分頁回應、Argument Resolver                              |
| host                  | 啟動層，負責組裝整個系統                     | Spring Boot 啟動類、模組載入、組態設定、安全設定、OpenAPI、環境設定                                         |
