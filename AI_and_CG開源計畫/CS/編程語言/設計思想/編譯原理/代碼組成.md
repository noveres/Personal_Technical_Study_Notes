**IDE（整合開發環境）** 包含了編輯器、除錯工具，以及最重要的核心——**編譯器（Compiler）或直譯器（Interpreter）**，正是它們負責執行這個「翻譯」工作 ，負責將我們寫的 Code 像是 Java 、C# 這類高級語言翻譯成機器碼，也就是由 0 跟 1 組成的機械語言

1. **Tokens** : 類似語言中的單詞，編譯器會先像「閱讀單詞」一樣，把你的原始碼（Source Code）切分成一個個最小的意義單位。就像讀英文時，把 "I love coding" 拆解成 "I", "love", "coding" ，編譯器也會將程式碼拆解成慣關鍵詞，標示符與字面量等....

範例
>`int age = 18;` 
>**Tokens**：
    - `int` (關鍵字 / Keyword)
    - `age` (標識符 / Identifier)
    - `=` (運算符 / Operator)
    - `18` (字面量 / Literal)
    - `;` (分隔符 / Separator)

2. **語法規則(Syntax)**：就是 **Tokens** 的組合規則，檢查句子結構是否正確。例如 "Coding love I" 雖然單詞對，但文法錯了。IDE 會在這一步抓出很多紅色的波浪底線（Syntax Error）
