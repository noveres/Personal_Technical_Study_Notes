>API ( Application Programming Interface )，中文叫應用程式介面，是一種提供不同軟體系統間互動的工具,定義了不同軟體間的互動規範
>

簡單來說 API 就是程式之間，約定好的一種調用方式，透過資訊的傳遞，允許不同的應用程式、服務或系統之間能夠共享資訊與功能,以約定好的 API 接口實現互聯互通。

API 是一個很廣的概念，意思是「程式之間溝通的方式」，包含程式**呼叫函式、作業系統提供的功能、庫與框架對外的接口(Interface)、AI Agent API (MCP)**，這些都是 API
